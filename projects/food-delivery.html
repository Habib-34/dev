Food Delivery App

Flutter • Firebase • Google Maps • Payment Gateway

Project Overview

A comprehensive food delivery application connecting customers with local restaurants. Features real-time order tracking, multiple payment options, restaurant discovery, and delivery management. Built with Flutter for optimal performance on both platforms.

Key Features

  • Restaurant discovery and menu browsing
  • Real-time order tracking with GPS
  • Multiple payment methods integration
  • Delivery time estimation
  • Rating and review system
  • Push notifications for order updates
  • Favorites and order history
