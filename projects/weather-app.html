Weather App

Flutter • Dart • REST API • Provider

Project Overview

A comprehensive weather application built with Flutter that provides real-time weather information, forecasts, and location-based weather data. The app features a clean, intuitive interface with smooth animations and responsive design.

Key Features

  • Real-time weather data from OpenWeatherMap API
  • 7-day weather forecast
  • Location-based weather detection
  • Search weather by city name
  • Beautiful weather animations
  • Dark/Light theme support
  • Offline data caching
