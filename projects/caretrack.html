CareTrack - Labor Management

Flutter • Firebase • GPS Tracking • Real-time Database

Project Overview

CareTrack is a comprehensive labor management and awareness application designed to improve workplace safety and worker rights. The app provides real-time tracking, safety monitoring, and educational resources for both workers and employers in the construction industry.

Key Features

  • Real-time GPS location tracking
  • Safety incident reporting system
  • Worker rights education module
  • Emergency contact and SOS features
  • Timesheet and attendance management
  • Multi-language support
  • Offline capability for remote sites
