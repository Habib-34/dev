Real-time Chat App

Flutter • Firebase • Socket.IO • Push Notifications

Project Overview

A modern real-time messaging application built with Flutter and Firebase. Features instant messaging, group chats, media sharing, and push notifications. Designed with a clean UI and smooth user experience for seamless communication.

Key Features

  • Real-time messaging with Socket.IO
  • Group chat functionality
  • Image and file sharing
  • Push notifications
  • User authentication with Firebase
  • Online/offline status indicators
  • Message encryption for security
