Angry Bird Game

Unity • C# • Physics Engine • Mobile Gaming

Project Overview

A physics-based mobile game inspired by the classic Angry Birds. Players launch birds using a slingshot to destroy structures and defeat enemies. Built with Unity engine featuring realistic physics, multiple levels, and engaging gameplay mechanics.

Key Features

  • Physics-based slingshot mechanics
  • 50+ challenging levels
  • Multiple bird types with unique abilities
  • Destructible environments
  • Score system and achievements
  • Sound effects and background music
  • Touch controls optimized for mobile
