E-commerce App

Flutter • Firebase • Stripe • REST API

Project Overview

A full-featured e-commerce mobile application with modern UI/UX design. Includes product browsing, shopping cart, secure payment processing, order tracking, and user account management. Built with Flutter for cross-platform compatibility.

Key Features

  • Product catalog with search and filters
  • Shopping cart and wishlist functionality
  • Secure payment integration with Stripe
  • Order tracking and history
  • User authentication and profiles
  • Push notifications for deals
  • Admin panel for inventory management
